# WEB2 Projekt 4: HTTP2

Struktura projekta bi trebala izgledati ovako:

```
nginx/
├── conf
    ├── nginx.conf*
    └── ...
├── html*
    ├── lab4*
        ├── index.html*
        ├── img*
        ├── css*
├── ssl*
    ├── localhost.crt*
    └── localhost.key*
```

zbog relativne putanje prema ssl certifikatima u nginx.conf datoteci.
Simbolom * su označene datoteke priložene uz ovaj repozitorij.
